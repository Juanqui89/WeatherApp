Aplicación del tiempo, diseñada con HTML, CSS y JavaScript, Responsive Design y utiliza la API de OpenWeather, para proporcionar información actualizada sobre las condiciones climáticas. El 
diseño atractivo y funcional creado con HTML y CSS ofrece una experiencia de usuario intuitiva y responsive en diversos dispositivos. A través de JavaScript, la aplicación interactúa con la 
API de OpenWeather para obtener datos en tiempo real, incluyendo temperatura, condiciones atmosféricas, velocidad del viento y pronósticos para una ubicación específica. La aplicación 
brinda a los usuarios la posibilidad de verificar el tiempo actual y futuros pronósticos de manera fácil y rápida. En resumen, esta aplicación del tiempo ofrece una interfaz amigable y 
precisa, utilizando tecnologías web estándar para proporcionar información climática actualizada de manera eficiente.
